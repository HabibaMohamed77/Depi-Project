# ☁️ DEPI Microsoft Data Engineer Program — Final Project

A comprehensive data engineering solution developed as the capstone project
for the Microsoft Data Engineer track under the DEPI initiative.

## 🚀 What This Project Does
- Ingests and processes real-world datasets using Azure cloud services
- Builds scalable ETL pipelines for data transformation
- Performs advanced SQL analytics on processed data

## 🛠️ Tech Stack
- **Python** — data processing & pipeline automation
- **Azure** — cloud storage, Data Factory, and compute services
- **SQL** — data transformation and analytics
- **Jupyter Notebook** — pipeline development & exploration
- **Power BI** — data visualization and dashboards

## 📌 Key Results
- Deployed a fully cloud-native data pipeline on Azure
- Processed and transformed large-scale datasets for analytics
- Delivered actionable insights through Power BI dashboards
